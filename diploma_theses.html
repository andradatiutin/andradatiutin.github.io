Diploma Theses

  • Guidelines

  • Above are attached a few guidelines for preparing your diploma thesis. The document contains as well information about the requirements for each phase listed in the "Timeline" section.

  • Timeline
    • 01 dec. 2023 - Topic Selection
            Required: One page document describing the selected topic
    • 30 jan. 2024 - Study and synthesis on the State of the Art (theoretical chapter)
            Required: Document - a short summary (2-4 pages) of existing work within your field of interest, including the used references
    • 15 feb 2024 - Application requirements. Short summary of research or application proposal
            Required: Document - short summary describing the topic in more detail, including identified requirements for your application
    • 15 mar 2024 Software application - initial version; Thesis structure + theoretical chapter
            Required: Software - an initial implementation of the software application. Document: thesis structure + theoretical chapter
    • 15 apr 2024 - Software application - alpha version; Application chapter - initial version
            Required: Software - a first functional version of the software application. Document: initial version of the application chapter
    • 01 jun 2024 - Software application - beta version; Application chapter - advanced version. Experiments chapter
            Required: Software a more advanced version of the software application. Document: application chapter - advanced version + experiments chapter
    • 15 jun 2024 - Software application - final version; Final version of the paper
            Required: Software - the final version of the software application. Document: the final thesis
